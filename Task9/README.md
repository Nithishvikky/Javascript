# Infinite Scrolling Content Loader

## Overview
  This page will load additional content as the user scrolls toward the bottom.

## New Things Learned
- Learned when It reached the bottom of the page. 
 -- **window.scrollY+window.innerHeight >= document.documentElement.scrollHeight**

- After reaching the bottom it will call the async function. Learned about the `async` and `await` topics.
